
Determine if a Sudoku is valid. 
The Sudoku board could be partially filled, where empty cells are filled with the number 0.

Sudoku rules:

The Sudoku is a number placing puzzle based on a 9x9 grid with several given numbers. 
The object is to place the numbers 1 to 9 in the empty squares so that each row, each column and each 3x3 box contains the same number only once.

Exemple:

-------------------------------------
| 5 | 3 |   |   | 7 |   |   |   |   |
-------------------------------------
| 6 |   |   | 1 | 9 | 5 |   |   |   |
-------------------------------------
|   | 9 | 8 |   |   |   |   | 6 |   |
-------------------------------------
| 8 |   |   |   | 6 |   |   |   | 3 |
-------------------------------------
| 4 |   |   | 8 |   | 3 |   |   | 1 |
-------------------------------------
| 7 |   |   |   | 2 |   |   |   | 6 |
-------------------------------------
|   | 6 |   |   |   |   | 2 | 8 |   |
-------------------------------------
|   |   |   | 4 | 1 | 9 |   |   | 5 |
-------------------------------------
|   |   |   |   | 8 |   |   | 7 | 9 |
-------------------------------------
